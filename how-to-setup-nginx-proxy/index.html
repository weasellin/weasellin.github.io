How to Setup Nginx Reverse Proxy

24 July 2017

The Nginx Reverse Proxy service could be started easily by the Nginx image.

With my configuration,

$ git clone https://github.com/weasellin/nginx_conf.git

Replace the ${HOST_IP} in the service_*.conf with the host ip

Since the Nginx would be run in the container, we have to explicitly assign host ip instead of using localhost.

To run the service,

$docker run \
    --name ansel-nginx \
    -p 80:80 \
    -d \
    --restart always \
    -v ${NGINX_CONF_DIR}:/etc/nginx/conf.d \
    nginx
