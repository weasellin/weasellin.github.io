How to Setup Jupyter PySpark Server

22 July 2017

From this image, it's easy to start a Jupyter Notebook Server with PySpark.

To preserve file and data, we have to prepare a host directory, ex.

$ export JUPYTER_DATA_DIR=${HOME}/service/jupyter_data
$ mkdir -p ${JUPYTER_DATA_DIR}
$ chmod 777 ${JUPYTER_DATA_DIR}

Notice that, there is a workaround to allow the privilege of notebook user in the container, by chmod for the ${JUPYTER_DATA_DIR}

By the following command, the Jupyter Lab Notebook server would be up in http://localhost:8888

$ docker run \
    --name ansel-jupyterlab \
    -p 8888:8888 \
    -d \
    --restart always \
    -v ${JUPYTER_DATA_DIR}:/home/jovyan \
    jupyter/pyspark-notebook \
    start.sh \
    jupyter lab \
    --NotebookApp.token='******'
