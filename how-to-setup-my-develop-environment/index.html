How to Setup Python pyenv

Follow this document to install required libraries:

$ sudo apt-get install -y make build-essential libssl-dev zlib1g-dev libbz2-dev libreadline-dev libsqlite3-dev wget curl llvm libncurses5-dev libncursesw5-dev xz-utils tk-dev

Then by this document to install pyenv:

$ curl -L https://raw.githubusercontent.com/pyenv/pyenv-installer/master/bin/pyenv-installer | bash

...

# Load pyenv automatically by adding
# the following to ~/.bash_profile:

export PATH="/home/weasellin/.pyenv/bin:$PATH"
eval "$(pyenv init -)"
eval "$(pyenv virtualenv-init -)"

Install some python versions to pyenv:

$ pyenv install --list
   ...
$ pyenv install 2.7.8
$ pyenv install 3.6.0
$ pyenv install pypy3-2.4.0
$ pyenv versions
* system (set by /home/weasellin/.pyenv/version)
  2.7.8
  3.6.0
  pypy3-2.4.0

Create virtual env from installed versions:

$ pyenv virtualenv 2.7.8 myenv-2.7.8
$ pyenv virtualenv 3.6.0 myenv-3.6.0
$ pyenv virtualenv pypy3-2.4.0 myenv-pypy3-2.4.0

Activate & deactivate virtual env:

$ pyenv activate myenv-3.6.0
$ pyenv deactivate

How to Setup Ghost

From this docker image, it's very easy to run a Ghost service by the command:

$ mkdir -p $GHOST_DATA_DIR
$ docker run \
      --name ansel-ghost \
      -p 2368:2368 \
      -d \
      -v $GHOST_DATA_DIR:/var/lib/ghost \
      ghost

Then, go to http://<domain>:2368/ghost/ to manage the Ghost blog.

How to Publish to GitHub Pages

Follow this tutorial to create a <username>.github.io repository.
Then, use the Buster tools to setup, generate, preview, and deploy the Ghost contents to the Github Pages.

$ cd $GHOST_DATA_DIR
# buster incompatible to python3
$ pyenv activate myenv-2.7.8
$ pip install buster
$ buster setup --gh-repo=git@github.com:<username>/<username>.github.io.git
$ buster generate
$ buster preview
$ buster deploy
Good job! Deployed to Github Pages.

After deployment, the Pages in https://<username>.github.io should be updated.
